1. Introducción a Java

JDK, JRE, JVM, compilación y primer programa en Java

¿Qué es Java?

Lenguaje de programación orientado a objetos, multiplataforma y de alto nivel.

✓ Multiplataforma

Escribe una vez, ejecuta en cualquier lugar (WORA)

✓ Orientado a Objetos

Código organizado en clases y objetos reutilizables

✓ Robusto y Seguro

Gestión automática de memoria y manejo de errores

Características Principales

Característica Descripción Beneficio para ASIR
Portabilidad Funciona en Windows, Linux, macOS Scripts multiplataforma
POO Programación Orientada a Objetos Código modular y mantenible
Bibliotecas Amplia API estándar Gestión de archivos, redes, BD
Multihilo Ejecución concurrente Automatización de tareas paralelas

JDK, JRE y JVM: Conceptos Clave

JDK
Java Development Kit
⊃
JRE
Java Runtime Environment
⊃
JVM
Java Virtual Machine

JDK

Para desarrollar
Compilador (javac), debugger, herramientas

JRE

Para ejecutar
JVM + bibliotecas estándar

JVM

Máquina virtual
Ejecuta bytecode Java

Primer Programa: "Hola Mundo"

``` // HolaMundo.java public class HolaMundo { public static void main(String[] args) { System.out.println(”¡Hola Mundo!”); } }

Salida
# Compilar $ javac HolaMundo.java # Ejecutar $ java HolaMundo ¡Hola Mundo!

Instalación del Entorno

1. Instalar JDK

Linux: sudo apt install openjdk-17-jdk
Windows: Descargar de Oracle/OpenJDK

2. Verificar Instalación

java --version
javac --version

3. Configurar PATH

Agregar directorio bin del JDK a la variable PATH del sistema

4. IDE (opcional)

Eclipse, IntelliJ IDEA, VS Code con extensiones Java

Conceptos Clave

  • Java es multiplataforma gracias a la JVM
  • El código fuente (.java) se compila a bytecode (.class)
  • JDK para desarrollar, JRE para ejecutar
  • Todo programa Java comienza en el método main()
  • Ideal para scripts de administración de sistemas
